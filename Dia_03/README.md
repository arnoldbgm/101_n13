## Como usar CSS
Para implementar CSS en un documento HTML, existen tres metodos, pero hoy conoceremos los dos mas comunes:
1. Usando la etiqueta `<style>` dentro del `<head>` del documento HTML.
```html
<head>
      <style>
         /* CSS va aqui */
         body {
               background-color: lightblue;
         }
         h1 {
               color: white;
               text-align: center;
         }
      </style>
</head>
```
